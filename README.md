# АКРОТЕРА — сайт-каталог изделий из архитектурного бетона

Статический сайт на чистом HTML / CSS / JS, без фреймворков и без привязки к конструкторам (Тильда и т.п.). Можно залить на любой обычный хостинг.

## Структура проекта

```
akrotera/
├── index.html          — вся вёрстка страницы (шапка, hero, материал, каталог, производство, CTA, футер, модалки)
├── css/
│   └── styles.css      — все стили (токены палитры/типографики в начале файла, :root)
├── js/
│   └── app.js           — логика: подгрузка товаров из JSON, переключение категорий, модалки заказа и карточки товара
├── data/
│   └── products.json    — ВСЕ товары, цены, размеры, цвета. Редактируется без программиста.
└── images/              — папка для фото (пока пустая — сейчас вместо фото плейсхолдер-иконка)
```

## Как добавить/изменить товар

Открываете `data/products.json`, находите нужную категорию по `id` (vases, sculptures, maf, fireplaces, sfb, thermopanels, facade-decor) и добавляете объект в массив `products`:

```json
{
  "id": "vase-new",
  "name": "Вазон новый",
  "dimensions": "Высота 40 см",
  "weight": "20 кг",
  "price": 12000,
  "colors": ["Белый", "Серый"]
}
```

Если `price` неизвестен — поставьте `null`, сайт сам покажет «По запросу» (так сделано для категорий СФБ, термопанели, фасадный декор — там цены индивидуальные).

Цвета берутся из словаря `colorSwatches` в том же файле — если нужен новый цвет, добавьте его туда с hex-кодом.

## Фото товаров

Все 19 товаров с фиксированной ценой (вазоны, скульптуры, маскароны, камины) подключены к реальным фото — итого 19 из 19. Файлы лежат в `images/` в формате `.jpg`, сжаты под веб (макс. сторона 1200px, около 2.4 МБ суммарно на все фото).

Товаров без фото нет — кроме категорий СФБ, термопанели и фасадный декор, где карточек вообще не предусмотрено (цена и комплектация там индивидуальные, сайт сразу показывает блок с пояснением).

### Как добавить ещё фото к товару

1. Положите файл в `images/`, например `images/mask-bacchus-2.jpg` (для второго ракурса).
2. В `data/products.json` у нужного товара расширьте массив `"images": ["images/mask-bacchus-1.jpg", "images/mask-bacchus-2.jpg"]`. Несколько фото — в карточке товара появится мини-галерея с переключением.
3. Больше ничего менять не нужно — `js/app.js` сам подставит фото в карточки каталога и в модалку.

## Форма заявки

Форма в модалке («Заказать расчёт» / «Отправить заявку») сейчас работает только на фронтенде — показывает сообщение об успехе, но никуда не отправляет данные. Для реальной работы нужно подключить один из вариантов:

- отправка на email (например, через сторонний сервис типа Formspree, либо свой backend);
- отправка в Telegram-бота (у вас уже есть опыт с Telegram-ботами — можно сделать так же);
- запись в Google Таблицу через Google Apps Script.

Это нужно дописать в `js/app.js`, в обработчике `orderForm.addEventListener("submit", ...)`.

## Палитра и шрифты

- Графитовый бетон `#1C1B19` / почти чёрный `#0E0D0C` — фон hero, футера, секции «Производство».
- Тёплый камень `#E8E2D6` / бумага `#F4F0E7` — светлые секции.
- Терракот/глина `#B5562F` — акцентный цвет (кнопки, цифры, эйбрау).
- Шрифты: Fraunces (заголовки, с засечками) + Inter (текст), оба подключены с Google Fonts.

## Запуск локально

Просто откройте `index.html` в браузере, либо поднимите локальный сервер:

```
python3 -m http.server 8000
```

и откройте `http://localhost:8000`.
